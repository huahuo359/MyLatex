# 浙江大学本科实验报告模板

by 雪松Cedar from CC98

套用的是学院某课程给的word模板，可根据需要在cls文件里自行调整样式。内附数学公式和表格的示例代码供参考。

用Word写实验报告排版令人糟心而且花时间，大学生应该把有限的时间投入到更有趣的事情中去。LaTeX就是一个让我们专心于内容创作的好工具，排版的事情就交给机器吧。

编译采用xelatex。
